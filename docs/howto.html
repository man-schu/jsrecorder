How to use JSRecorder

Press "Record Audio" to record audio only, using the standard microphone.

Press "Record Video" to record video only, using the standard camera.

Press "Record Screen" to record the full screen or windows of your choice.

In order to be able to use this application, you have to enable the corresponding devices.

For instance, to record a video you have to allow camera and microphone:

Once you are finished, press "Stop".

You will be asked to enter a file name:

Press OK.

Now you will be shown a button "Save as webm file".

This will allow you to save the recording as a file.

Note that this application cannot store in a compressed format, as it is fully running on the local machine. The advantage is that there is no privacy issue as there would be if it ran on a server.
